Product Revision or Latest Product Version Information.


In Normal cases we need to manage different versions of same product. To handle such cases we add new products in odoo which is perfect way to manage but in that we can not get which product is latest and which one is older version. To handle this scenario we have added this module where you can set previous version of product.
On Product and Product template both views previous version and latest version information can be Managed.

Manage Information about Previous Version of Product.
Information about all new latest version of the product.

Other Configuration

To use this module no technical knowledge is required even no configuration is needed. Just install module and functionality is ready to use. For more customization and help contactus.
