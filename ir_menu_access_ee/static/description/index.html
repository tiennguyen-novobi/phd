Odoo 13 Security for Menu

Allow user to hide menu for particular users or groups.

Note: This module is tested and working in Enterprise Edition. And we are supporting Ubuntu OS and not Windows OS.

Available for Community

How to use

  • For using the module, make configuration into 'Settings - Technical - Security - Menu Access Security' menu. Select menus and users or groups for hiding the menus from users.
  • Select menu(s) that we want to hide and then select group(s) or user(s) for whom we want to hide that selected menu(s).
  • Now, save and Refresh browser, selected menu(s) from will removed from the menu(s).
